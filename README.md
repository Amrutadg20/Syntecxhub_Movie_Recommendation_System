# Movie Recommendation System

## Project Overview

This project is a Content-Based Movie Recommendation System developed using Python and the MovieLens dataset.

The system recommends movies similar to a selected movie by analyzing movie genres and calculating similarity scores using Cosine Similarity.

## Features

* Movie recommendation based on genres
* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature extraction using CountVectorizer
* Cosine Similarity based recommendations
* Simple and beginner-friendly implementation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## Dataset

MovieLens Dataset

Files used:

* movies.csv
* ratings.csv

## Project Workflow

1. Load Dataset
2. Perform Data Cleaning
3. Conduct Exploratory Data Analysis
4. Extract Features from Movie Genres
5. Calculate Cosine Similarity
6. Generate Movie Recommendations

## Sample Recommendation

Input:
Toy Story (1995)

Output:

* Toy Story 2
* Monsters Inc
* Finding Nemo
* Cars
* Shrek

## Conclusion

This project demonstrates the fundamentals of recommendation systems using content-based filtering techniques. It can be further enhanced by implementing collaborative filtering and deploying it as a web application.
